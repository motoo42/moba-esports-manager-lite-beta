# 구현 순서 안내판

이 파일은 현재 스프린트의 방향만 빠르게 보기 위한 임시 안내판

전체 체크리스트와 완료 여부는 [docs/development-checklist.md](./docs/development-checklist.md)를 기준으로 관리
새 계정이나 새 에이전트로 이어받을 때는 [CODEX_HANDOFF.md](./CODEX_HANDOFF.md)를 먼저 확인

## 현재 위치

- 기준일: 2026-06-08
- 현재 진척도 추정: 1시즌 MVP 기준 완료, 기말 프로젝트 LCK 3시즌 작동 기준 100% 내외, 정교한 개인 프로젝트 v1 기준 약 54~62%
- 최우선 목표: 기말 프로젝트 LCK 3시즌 작동
- 목표 완료 기준: 2026, 2027, 2028 시즌을 진행하고 2028 Worlds 종료 후 시즌 요약/스토브리그까지 안정적으로 도달해 2029 진입 직전 상태를 확인
- 최근 완료: 베타 피드백 A 묶음(스토브리그 제안 역할, 영입 확정 대기, 예산/포지션 제한, 우리 팀 이적 로그 하이라이트), 베타 피드백 B 묶음(LCK 10팀 선택, Ghost SUP 수정, 로스터 조작 즉시 사기 변화 제거), 베타 전 재정비 8번 링크 하나로 접속 가능한 배포 구조 전환, 베타 전 재정비 3번 1군/2군 로스터 분리와 콜업/콜다운, 베타 전 재정비 2번 2026 시작 전 프리시즌 스토브리그, 베타 전 재정비 1번 리렌더링/Route 왕복 버그 수정, 좌측 메뉴 그룹형 한글 사이드바 개선, 데이터 저장 메뉴 분리 UI, UI 검토와 화면 polish 1차, 2026 LCK 1군 선수 사진 구조/리소스 적용, 모바일 미지원 정책, LCK 팀 밸런싱 1차, 안정성 하드닝, 저장 안정성/마이그레이션 최소 보강, 시즌 히스토리/3시즌 결산 UI
- 0번 작업 완료: 기말 프로젝트 LCK 3시즌 목표 문서화
- 1번 작업 완료: 2026 LCK 10개팀 1군+2군 전체 선수 데이터 import
- 2번 작업 완료: 2026 아시안게임 시즌과 2027/2028 일반 시즌 프로필 분기
- 3번 작업 완료: 일반 시즌 LCK Rounds 3-5 실제 경기 구현
- 3.5번 작업 완료: 사용자 능력치 세부조정 1차
- 4번 작업 완료: 2027 일반 시즌을 LCK Cup부터 Worlds/오프시즌/2028 LCK Cup 진입까지 debug runner로 관통 검증
- 6번 작업 일부 완료: 성장/하락/시장가치/은퇴 후보 플래그 적용. 유망주 생성과 실제 은퇴 처리는 후속
- 다음 목표: 닫힌 스토브리그 정보 화면 또는 타팀 로스터/스카우팅/선수 카드/메시지함 묶음
