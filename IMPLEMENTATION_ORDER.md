# 구현 순서 로드맵

이 파일은 남은 구현 순서를 빠르게 확인하기 위한 임시 안내판입니다. 전체 기능 구현이 마무리되어 더 이상 필요 없으면 삭제해도 됩니다.

작업 기본 흐름:

1. UI 설계 컨펌
2. 도메인 로직 구현
3. 화면 연결
4. 테스트
5. 16:9 캡처 저장
6. 체크리스트 갱신

## 현재 위치

- 현재 체크리스트 기준 진척도: 약 55.2%.
- 다음 작업: `11-B. LCK Rounds 1-2 포스트시즌 구현`.
- 큰 목표: LCK Cup 이후의 국내 시즌, 국제전, 시즌 종료, 다음 시즌까지 이어지는 1시즌 완주 루프 완성.
