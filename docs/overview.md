# 롤게임 후보 개요

상태: 프로젝트 주제 후보 초안  
유형: 가상 e스포츠 커리어 매니지먼트 시뮬레이션  
런타임 AI 의존도: MVP에서는 없음  
주요 리스크: 20시즌 장기 커리어 스코프, 시뮬레이션 밸런스, IP 안전성, 커스텀 스킨 데이터 검증

## 1. 콘셉트

사용자가 가상의 5인 MOBA 팀을 구성하고 최대 20시즌의 커리어를 운영하는 웹 기반 매니지먼트 시뮬레이션입니다.

실제 League of Legends 선수, 팀, 로고, Riot 에셋을 쓰지 않고, e스포츠 매니지먼트에서 영감을 받은 가상 게임으로 설계합니다.

기획 기준으로는 LCK형 리그를 기본 무대로 삼습니다. 사용자는 국내 리그 선수 풀 안에서만 로스터를 구성하고, 국제 대회 시점에만 미리 설정된 해외 팀/해외 선수들이 상대 팀으로 등장합니다. 배포용 기본 데이터에서는 실제 명칭 대신 가상 리그/가상 팀명을 사용합니다.

핵심 경험:

```text
국내 로스터 구성 -> 전략/훈련 설정 -> 국내전 진행 -> 국제전 진출 -> 해외 팀 상대 -> 장기 커리어 관리 -> 20시즌 완주
```

## 2. 한 줄 요약

사용자가 가상의 MOBA 선수들을 영입하고, 팀 케미/전략/훈련/계약을 관리하며 최대 20시즌 커리어를 플레이하는 e스포츠 매니저 게임입니다.

## 3. 타깃 사용자

주요 사용자:

- e스포츠 팬.
- 전략 게임을 좋아하는 사용자.
- 로스터 구성과 팀 운영을 즐기는 사용자.

보조 사용자:

- MOBA의 밴픽/팀 조합에 관심 있는 플레이어.
- 복잡한 풀 스케일 시뮬레이터보다 짧고 가벼운 매니지먼트 게임을 원하는 사용자.

## 4. 핵심 사용자 가치

사용자는 e스포츠 팀을 직접 구성하고 운영하며, 단기 시즌 성과와 장기 커리어 운영을 함께 경험할 수 있습니다.

앱은 로스터 구성, 전략 선택, 훈련, 선수 교체, 상대 분석이 실제 승패와 장기 팀 운영에 의미 있게 연결되는 전략 게임이어야 합니다.

## 5. 주요 확정 결정

- 최종 목표는 최대 20시즌 플레이 가능한 커리어 모드입니다.
- 궁극적으로는 로스터 구성과 플레이를 합쳐 한 시즌 20시간 정도 걸리는 깊이를 원합니다.
- 간소화 버전은 한 시즌 2-3시간 정도를 목표로 합니다.
- 국제전은 조별리그 이후 싱글 엘리미네이션 토너먼트로 진행합니다.
- 국제전 포맷은 축구 월드컵식 구조를 참고합니다.
- 사용자는 로스터 설정, 전략, 훈련, 선수 교체를 직접 결정합니다.
- 경기 결과는 팀 종합 스탯을 ELO 점수로 표현해 계산합니다.
- 선수는 나이를 먹고 은퇴합니다.
- 은퇴/커리어에는 군 입대 이슈도 고려합니다.
- 유망주는 새롭게 생성될 수 있습니다.
- 계약은 1년, 2년, 1+1년 옵션만 우선 고려합니다.
- 아시안게임 로스터는 시스템이 지정합니다.
- 해외 팀은 사전 정의된 데이터셋을 사용합니다.
- 현실성 한계는 감수하고, 해외 리그당 4개 팀 정도를 고정 데이터로 둡니다.
- 사용자의 주요 실력 개입 지점은 로스터, 전략 선택, 상대 분석입니다.
- 꼴찌 시즌 보상이나 리빌딩 보상은 추후 구현으로 둡니다.
