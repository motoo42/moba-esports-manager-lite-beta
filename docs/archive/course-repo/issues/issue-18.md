## 문제
현재 내 예산이 얼마나 남았는지, 기존 총 예산이 얼마였는지 확인하기 어렵다.

## 기대 동작
- 로스터 관리 화면에서 우리 팀 전체 선수 연봉 총합을 명확히 보여준다.
- 스토브리그 화면에서 총 예산, 현재 사용 연봉, 잔여 예산을 명확히 보여준다.
- FA 영입 확정/재계약 협상 시 예산 변화를 예측할 수 있어야 한다.

## 완료 기준
- 로스터 관리/스토브리그 화면에 예산 요약 패널이 추가된다.
- 금액 표기는 기존 1 = 1천만원 기준 한국식 금액 포맷을 사용한다.
- 예산 초과 상태가 시각적으로 명확히 표시된다.
