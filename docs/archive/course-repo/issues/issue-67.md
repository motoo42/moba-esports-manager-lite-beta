## 배경

60번/65번 작업으로 홈 화면에 선발 5인 로스터와 팀 로고 정렬 개선이 들어갔지만, 매치 허브 전체 UI는 아직 장기적으로 더 다듬을 여지가 있습니다.

현재 홈 화면은 주요 정보가 한 화면에 모이는 장점이 있지만, 매치 프리뷰, 오늘 일정, 선발 5인, 상대 분석, 최근 메시지, 진행 상태가 함께 배치되면서 화면 밀도와 세로 흐름을 다시 정리할 필요가 있습니다.

## 요구사항

- 홈 화면의 매치 허브 UI와 레이아웃을 전반적으로 재검토합니다.
- 매치 프리뷰, 오늘 일정, 선발 5인, 상대 분석, 최근 메시지, 진행 상태의 우선순위를 다시 정합니다.
- 홈 화면에서 바로 확인해야 하는 정보와 세부 화면으로 보내도 되는 정보를 구분합니다.
- 선발 5인 로스터는 홈에서 확인되는 맛을 유지하되, 사진/이름/별점이 잘리거나 화면 밖으로 밀리지 않게 합니다.
- 선발 5인 카드는 현재처럼 세로형 카드의 시각적 느낌을 우선 유지합니다.
- compact 가로형 카드는 기본 방향으로 삼지 않습니다.
- 오늘 일정 카드가 좌우로 과하게 길게 늘어지지 않도록 정보 밀도와 카드 폭을 재정리합니다.
- 매치 허브 전체의 세로 길이, 여백, 카드 간격을 다시 조정합니다.
- 데스크톱 첫 화면에서 핵심 정보가 자연스럽게 보이는지 확인합니다.
- 모바일/좁은 화면에서도 카드가 겹치거나 잘리지 않게 합니다.

## 완료 기준

- 홈 화면 매치 허브의 주요 정보가 더 읽기 쉬운 구조로 정리됩니다.
- 선발 5인 로스터 카드의 사진, 이름, 별점이 안정적으로 보입니다.
- 오늘 일정과 상대 분석 영역이 과하게 넓거나 길게 느껴지지 않습니다.
- 홈 화면의 첫 인상이 번잡하지 않고, 플레이 직전 상태를 빠르게 이해할 수 있습니다.
- 데스크톱/모바일에서 레이아웃 깨짐이 없습니다.

## 우선순위

후순위 장기 개선 작업입니다.
