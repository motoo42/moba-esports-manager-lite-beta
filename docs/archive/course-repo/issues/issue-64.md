## Body

## 배경

현재 전략/훈련 탭에서 [전략] 하위 메뉴를 누르면 주간 계획 메뉴의 아래쪽으로 바로 이동하는 구조입니다. 이 방식은 전략 조정 화면이 독립적인 메뉴라는 느낌을 주지 못하고, 사용자가 전략을 조정하는 흐름도 명확하지 않습니다.

## 요구사항

- [전략] 하위 메뉴를 주간 계획 내부 앵커가 아니라 독립 화면으로 분리합니다.
- 전략을 짜는 전용 화면을 만들고, 그 안에서 전략을 조정할 수 있게 합니다.
- 주간 계획 화면과 전략 조정 화면의 역할을 분명히 나눕니다.
- 전략 변경 결과는 기존 경기/상대 분석/주간 계획 데이터와 계속 연결되어야 합니다.

## 완료 기준

- [전략] 버튼을 눌렀을 때 독립된 전략 조정 화면으로 이동합니다.
- 주간 계획 화면의 아래쪽으로 단순 스크롤 이동하는 구조가 사라집니다.
- 전략 선택/변경 UI가 전용 화면 안에서 명확하게 보입니다.
- 기존 전략 저장 및 경기 시뮬레이션 결과와 충돌하지 않습니다.
