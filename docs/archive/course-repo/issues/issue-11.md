## 배경

주요 기능 구현 후에는 실제 플레이를 반복하면서 버그, UX 문제, 밸런스 문제를 찾아야 한다.  
특히 베타 테스트 전에 2026 프리시즌부터 시즌 진행, 저장/불러오기, 스토브리그, 로스터 관리가 자연스럽게 이어지는지 확인해야 한다.

## 작업 내용

- 새 커리어 생성부터 2026 프리시즌 스토브리그 진행 테스트
- 재계약 / 방출 / FA 영입 / 영입 확정 흐름 점검
- 1군 / 2군 로스터 분리와 콜업 / 콜다운 점검
- 경기 진행과 날짜 진행 점검
- 대회 현황, 캘린더, 시즌 결산 화면 점검
- 저장 / 불러오기 / 자동 저장 점검
- Render 배포 링크에서 실제 접속 테스트
- 모바일 미지원 안내가 의도대로 보이는지 확인
- 발견한 문제를 우선순위별로 정리
- 치명적인 진행 불가 버그 우선 수정

## 완료 기준

- 새 커리어 시작 후 주요 흐름이 끊기지 않는다.
- 저장/불러오기 후에도 진행 상태가 유지된다.
- 베타테스터에게 링크를 보내도 기본 플레이가 가능하다.
- 발견한 버그 목록과 수정 여부가 문서화된다.
- `npm.cmd test`, `npm.cmd run build` 통과
