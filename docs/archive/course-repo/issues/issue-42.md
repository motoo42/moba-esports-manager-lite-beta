## Body

﻿## 배경
현재 저장 데이터는 개별 삭제만 가능하다. 테스트를 반복하면 저장 슬롯이 많이 쌓이므로, 여러 저장 데이터를 한 번에 정리할 수 있는 관리 기능이 필요하다.

## 요구사항
- 데이터 저장 메뉴에 `일괄 삭제` 버튼을 추가한다.
- 버튼 클릭 시 별도 모달/창에서 현재 저장된 시즌 데이터 목록을 보여준다.
- 각 저장 슬롯은 체크박스로 선택할 수 있어야 한다.
- 선택한 저장 슬롯을 한 번에 삭제할 수 있어야 한다.
- 삭제 전 최종 확인 단계를 둔다.
- 삭제 성공/실패 결과를 저장 화면 안에서 명확히 안내한다.

## 완료 기준
- 여러 저장 슬롯을 선택해 한 번에 삭제할 수 있다.
- 삭제 취소 시 어떤 API 변경도 발생하지 않는다.
- 현재 플레이 중인 커리어 저장 슬롯이 삭제되는 경우에도 앱이 깨지지 않는다.
- 개별 삭제 기능과 일괄 삭제 기능이 함께 안정적으로 동작한다.
