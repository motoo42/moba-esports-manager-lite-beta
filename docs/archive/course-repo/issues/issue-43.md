﻿## 배경
현재 이적시장은 선수의 기존 지위와 제안받는 역할의 차이를 충분히 반영하지 못한다. 현실적으로 1군 주전 선수가 2군 제안을 받으면 대부분 거절하고, 2군 선수가 1군 제안을 받으면 낮은 금액에도 긍정적으로 반응할 가능성이 높다.

## 요구사항
- 선수의 기존 지위와 제안 역할을 비교해 최소 수락선을 다르게 계산한다.
  - 기존 1군 주전 -> 2군 제안: 강한 거절 경향, 최소 수락선 상승
  - 기존 1군 주전 -> 식스맨 제안: 소폭 거절 경향, 최소 수락선 상승
  - 기존 2군 -> 1군 주전 제안: 강한 긍정 반응, 최소 수락선 하락
  - 기존 2군 -> 식스맨 제안: 긍정 반응, 최소 수락선 하락
- 구현 난도가 높다면 최소한 위 케이스를 강한 제약조건으로라도 반영한다.
- 협상 분위기에도 역할 변화 효과가 즉시 반영되어야 한다.
- AI 제안과 유저 제안의 수락선 계산은 서로 독립적으로 유지한다.

## 완료 기준
- 1군 주전에게 2군 제안을 하면 웬만해서 수락하지 않는다.
- 2군 선수에게 1군/식스맨 제안을 하면 같은 금액이라도 분위기가 좋아진다.
- 역할 차이에 따른 수락선 변화가 테스트로 검증된다.
- 협상 UI에서 역할 변경 시 분위기 변화가 즉시 보인다.
