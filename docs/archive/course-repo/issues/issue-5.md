## 목표

선수 카드에서 사진, OVR, 이름, 포지션이 한눈에 보이도록 리디자인한다.

현재 일부 카드에서 OVR이 작게 보이고, 선발 슬롯 카드가 잘려 보이는 문제가 있다. 로스터 관리와 스토브리그의 핵심 경험이 선수 카드이므로 시각적 개선이 필요하다.

## 작업 내용

- FIFA 게임 형태 선수 카드 레이아웃 적용
  - 선수 사진 중심
  - OVR 크게 표시
  - 이름 명확히 표시
  - 포지션/나이/팀 정보 보조 표시
- 선발 슬롯 카드 잘림 문제 수정
- 후보 카드, 2군 카드, 스토브리그 카드, 선수 상세 모달 스타일 통일
- 사진이 없는 선수는 fallback 아바타 사용
- 카드 크기와 aspect-ratio 고정으로 레이아웃 흔들림 방지

## 완료 기준

- 로스터 관리 화면의 선발 슬롯 카드가 잘리지 않음
- 후보/2군/스토브리그 선수 카드에서 OVR과 이름이 잘 보임
- 선수 사진이 있는 경우 자연스럽게 표시되고, 없는 경우 fallback이 깨지지 않음
