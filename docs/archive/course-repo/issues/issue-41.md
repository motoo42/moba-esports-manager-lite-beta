﻿## 배경
현재 메시지함에 들어오는 메시지는 제목에 너무 많은 정보가 포함되어 있어, 목록/미리보기 화면에서 제목이 길게 잘리거나 핵심을 빠르게 파악하기 어렵다.

## 요구사항
- 메시지 제목은 핵심만 간결하게 표시한다.
- 상세한 날짜, 경기/이적/훈련 맥락, 추가 설명은 본문으로 이동한다.
- 미리보기 화면에서 제목이 잘리는 일이 드물도록 제목 길이 기준을 잡는다.
- 메시지 카테고리별 제목 작성 규칙을 정리한다.
  - 일정: `다음 경기 일정 안내`
  - 경기: `경기 결과 도착`
  - 이적: `FA 협상 결과`
  - 훈련: `훈련 상태 보고`
  - 시스템: `저장 상태 안내`

## 완료 기준
- 메시지 목록에서 긴 제목 때문에 UI가 흔들리지 않는다.
- 제목만 봐도 메시지 종류와 핵심을 알 수 있다.
- 자세한 내용은 본문에서 확인할 수 있다.
- 기존 저장 데이터의 긴 제목 메시지도 안전하게 표시된다.
