## 문제
로스터 관리 탭의 선발 5인 화면과 스토브리그 화면에서, 선수 카드를 클릭하지 않은 기본 상태일 때 모든 선수 평가가 별 5개로 표시된다. 클릭 후 상세 화면에서는 실제 평가 별점이 표시된다.

## 기대 동작
- 기본 카드와 상세 모달 모두 같은 평가 계산 결과를 표시한다.
- 로스터/스토브리그/허브/구단 정보 등 모든 카드에서 평가 표시가 일관되어야 한다.

## 완료 기준
- 기본 카드의 평가 별점이 실제 선수별 평가값을 반영한다.
- 스토브리그 선수 카드에서도 동일하게 반영된다.
- OVR/POT 직접 노출 없이 평가 별점만 표시된다.
- 회귀 테스트가 추가된다.
