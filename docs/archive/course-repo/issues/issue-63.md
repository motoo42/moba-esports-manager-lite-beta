## 배경

여러 대회 화면의 하위 메뉴와 토너먼트 표시 방식이 아직 일관되지 않습니다. 일부 버튼이 영어로 남아 있고, 브래킷 배치가 하위 2경기의 가운데에 상위 경기가 놓이는 토너먼트 구조로 보이지 않는 문제가 있습니다.

## 요구사항

- 대회 하위 메뉴 중 영어로 표시된 버튼을 전부 LCK Cup처럼 한글로 변경합니다.
- `Bracket` 또는 브래킷으로 표시되는 영역은 [토너먼트]로 통일합니다.
- 더블 엘리미네이션이든 월드컵식 토너먼트든 사용자-facing 명칭은 [토너먼트]를 기본으로 합니다.
- LCK 3-4 라운드의 토너먼트 영역이 현재 [진출 경로]로 잘못 표시되어 있다면 [포스트시즌]으로 변경합니다.
- LCK 3-5 라운드 포맷도 같은 규칙을 적용해 검토하고 수정합니다.

## 브래킷 배치 개선 대상

- LCK 1-2 라운드
- MSI
- LCK 3-4 라운드
- LCK 3-5 라운드
- 그 외 토너먼트/포스트시즌 구조를 표시하는 모든 대회 화면

## 브래킷 배치 요구사항

- 하위 2경기의 가운데 위치에 상위 경기가 배치되는 옆으로 누운 피라미드 구조가 나타나야 합니다.
- 현재 이 구조가 보이지 않는 대회는 전부 수정합니다.
- 토너먼트를 배치하는 그리드에 위아래 여유 공간이 많다면, 세로 여백을 줄여 화면 길이를 줄입니다.
- 여유 공간이 많지 않은 경우에는 기존 간격을 유지해도 괜찮습니다.
- 이 검토 및 수정은 모든 대회의 토너먼트 화면에 대해 수행합니다.

## 완료 기준

- 대회 하위 메뉴의 영어 버튼이 한글로 정리됩니다.
- 브래킷/Bracket 명칭이 사용자 화면에서 적절한 한글 명칭으로 통일됩니다.
- 각 대회의 토너먼트 구조가 시각적으로 자연스럽게 연결됩니다.
- 토너먼트 화면의 불필요한 세로 여백이 가능한 범위에서 줄어듭니다.
