## 문제

## 기대 동작
- LCK 구단 정보에는 구단 목록 메뉴만 남긴다.
- 특정 팀의 상세 화면 안에서 선발/후보/아카데미 섹션을 보여준다.

## 완료 기준

- /teams와 /teams/:teamId 접근은 유지한다.
