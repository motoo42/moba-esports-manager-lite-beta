## 목표

Football Manager의 메시지함처럼, 하루가 지날 때마다 중요한 정보를 알림으로 받을 수 있는 시스템을 만든다.

현재는 시즌 진행 결과나 계약 결과를 화면별로 찾아봐야 한다. 플레이어가 다음에 무엇을 해야 하는지 자연스럽게 알 수 있도록 메시지함/뉴스/일정 알림이 필요하다.

## 작업 내용

- 좌측 메뉴의 `중요 알림 / 뉴스 / 일정 알림` 구조를 실제 화면으로 구현
- 하루 진행 시 0개 이상의 메시지 생성
- 1차 메시지 종류:
  - 다음 경기 일정
  - 경기 결과 요약
  - 선수 폼/피로도/사기 변화
  - 훈련 반응
  - 재계약/FA 협상 결과
  - 우리 팀 이적 로그
  - 주요 AI 이적 소식
  - 대회 시작/종료 알림
- 특히 스토브리그의 `isUserTeamRelated` 이적 로그는 메시지함에도 노출
- 메시지 읽음/안읽음 상태는 1차에서는 단순 구현 또는 후속으로 분리 가능

## 완료 기준

- 하루 진행 후 메시지함에 관련 알림이 쌓임
- 우리 팀 계약/영입/방출/취소 로그가 메시지함에 표시됨
- 다음 경기나 대회 일정 같은 중요한 정보를 메인 화면 밖에서도 확인 가능
- 메시지함이 없어도 진행은 가능하지만, 있으면 다음 행동이 더 명확해짐
