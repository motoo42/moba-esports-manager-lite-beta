## Body

## 배경

팀 상세 화면에서 `1군 후보` 항목에 선수가 1명만 있을 때 카드가 좌우로 과도하게 늘어지는 레이아웃 문제가 다시 발생했습니다.

## 문제 상황

- `1군 후보`가 독립 영역으로 표시되면서 단일 선수 카드가 가로로 길게 늘어납니다.
- 이전에 유사한 레이아웃 문제를 수정한 적이 있으나, 현재 팀 상세 탭에서 다시 재현됩니다.
- 후보 선수를 보기 위한 별도 영역이 UI를 복잡하게 만들고, 아카데미 선수 목록과 역할이 겹칩니다.

## 요구사항

- 별도의 `1군 후보` 영역 또는 탭은 제거합니다.
- 기존 아카데미 선수 목록 영역을 재활용해 `1군 후보`와 `아카데미`를 같은 후보 선수 영역에서 전환할 수 있게 구성합니다.
- 1군 후보가 1명이어도 카드가 과도하게 늘어나지 않도록 카드 폭과 그리드 제약을 안정화합니다.
- 모바일/데스크톱 양쪽에서 카드가 겹치거나 늘어지지 않도록 확인합니다.

## 완료 기준

- 팀 상세 화면에서 독립된 `1군 후보` 영역이 사라집니다.
- 후보 선수 영역 안에서 `1군 후보`와 `아카데미`를 구분해 볼 수 있습니다.
- `1군 후보` 선수가 1명뿐이어도 카드가 보기 좋은 폭을 유지합니다.
- 관련 통합 테스트 또는 화면 검증이 추가/수행됩니다.
