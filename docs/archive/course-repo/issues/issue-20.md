## 문제
선수 클릭 시 나오는 세부 화면이 너무 작다. Football Manager처럼 선수 커리어와 설명을 볼 수 있는 큰 상세 화면이 필요하다.

## 기대 동작
- 선수 상세 화면은 현재보다 큰 대형 모달로 표시된다.
- 전체 화면을 완전히 덮지는 않되, 화면 대부분을 활용한다.
- 우상단 X 버튼으로 닫을 수 있다.
- 선수 커리어 기록을 표시한다. 예: T1 (2025.01.11 ~ 2026.12.27) 형태.
- 유명 선수는 간단한 설명/프로필 텍스트를 표시할 수 있는 구조를 둔다.

## 완료 기준
- 공통 선수 상세 모달 크기와 레이아웃이 재정의된다.
- 커리어 히스토리 데이터 구조 또는 표시 fallback이 추가된다.
- 로스터/스토브리그/구단 정보에서 같은 상세 화면을 재사용한다.
