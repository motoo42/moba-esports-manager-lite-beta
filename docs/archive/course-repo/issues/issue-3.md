## 목표

스토브리그 기간이 아닐 때도 `스토브리그` 메뉴에 접근할 수 있게 만든다.

현재는 스토브리그가 열려 있지 않으면 메뉴 접근/정보 확인이 제한적인데, 플레이어 입장에서는 FA 시장, 다음 스토브리그 일정, 최근 이적 로그를 언제든 확인할 수 있어야 한다.

## 작업 내용

- 스토브리그 기간이 아닐 때도 `/offseason` 또는 스토브리그 메뉴 접근 허용
- 시장이 닫혀 있으면 계약 액션 대신 정보 화면 표시
- 표시할 정보:
  - 다음 스토브리그 예정 시점
  - 현재 FA 선수 목록
  - 최근 이적 로그
  - 내 팀 계약 만료 예정자
  - 스토브리그 규칙 요약
- 실제 계약 제안/재계약/방출 버튼은 시장 기간에만 활성화

## 완료 기준

- 스토브리그가 닫힌 상태에서도 메뉴 클릭 시 빈 화면이나 접근 차단이 발생하지 않음
- 정보 화면에서 FA/계약/이적 로그를 확인할 수 있음
- 시장이 열렸을 때는 기존 스토브리그 진행 화면으로 정상 전환됨
