## 문제
시즌 결산도 데이터 저장처럼 하위 메뉴가 필요하지 않다. 대신 좌측 대형 메뉴 중 하나로 설정 메뉴를 추가하고 활용 방식을 고민할 필요가 있다.

## 기대 동작
- 시즌 결산 하위 메뉴를 제거한다.
- 좌측 메뉴에 설정을 추가한다.
- 설정 메뉴에 넣을 항목을 구체화한다. 예: 화면/저장/베타 정보/플레이 옵션 등.

## 완료 기준
- 시즌 결산 하위 메뉴가 제거된다.
- 설정 메뉴 route 또는 placeholder 화면이 추가된다.
- 설정 메뉴의 1차 범위가 문서화된다.
