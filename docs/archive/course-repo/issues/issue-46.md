﻿## 배경
현재 스토브리그는 유저가 1군 로스터와 2군 로스터를 모두 예산 안에서 직접 맞춰야 해, 진행이 막히거나 2군 영입에 지나치게 많은 신경을 써야 하는 문제가 있다. 베타 단계에서는 유저가 1군 로스터 완성에 집중하고, 2군은 선택적으로 구성할 수 있게 하는 편이 좋다.

## 요구사항
- AI는 기본적으로 2군 로스터용 선수 영입을 하지 않는다.
- AI는 기존 2군 선수라도 1군/식스맨으로 쓸 의도가 있을 때만 영입을 시도한다.
- 유저는 2군 선수를 2군 역할로 영입할 수 있지만, 이 경우 AI 경쟁은 발생하지 않는다.
- 유저가 반드시 2군 5명을 직접 영입해야 하는 조건을 제거한다.
- 스토브리그 종료 시 남아 있는 2군/아카데미 FA를 각 팀에 자동 배치한다.
- 자동 배치는 모든 팀이 최소 `1군 5인 + 2군 5인`을 맞추면 종료한다.
- 자동 배치되지 못한 2군 선수는 FA 상태로 남는다.
- 자동 배치 과정은 예산을 소비하지 않는다.
- 유저가 1군 4명 이하로 막힌 경우, 남은 2군 FA를 긴급 등록해 최소 1군 5인을 맞추는 fallback을 둔다.

## 완료 기준
- 유저는 2군 영입 없이도 스토브리그 종료가 가능하다.
- 스토브리그 종료 시 최소 로스터 기준이 자동으로 보정된다.
- 예산 부족 때문에 2군 로스터를 못 채워 진행이 막히는 일이 사라진다.
- 자동 배치 로그가 남아 어떤 선수가 어느 팀에 배치되었는지 확인할 수 있다.
- AI의 2군 영입 경쟁이 불필요하게 발생하지 않는다.
