## 문제
전략/훈련 시스템의 영향도가 체감되기 어렵고, 좌측 하위 메뉴인 전략, 훈련 강도가 클릭되지 않는다.

## 기대 동작
- 전략과 훈련 강도가 경기력/선수 상태에 약간이라도 체감 가능한 영향을 준다.
- 전략, 훈련 강도 하위 메뉴가 클릭 가능하다.
- 각 하위 메뉴에 어떤 정보를 넣을지 구체화한다.

## 완료 기준
- 전략/훈련 하위 메뉴 클릭 시 해당 섹션 또는 하위 화면으로 이동한다.
- 전략별 효과와 훈련 강도 효과가 UI에 설명된다.
- 경기 시뮬레이션 또는 선수 상태 변화에서 영향도가 테스트 가능하다.
