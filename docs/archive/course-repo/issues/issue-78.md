## 개요
블랙 모드, 화이트 모드, 시스템 설정 기반 모드 등 디자인 테마 전환 기능을 도입한다.

## 주요 요구사항
- 다크/라이트/시스템 기본값 모드 제공
- 현재 CSS 색상과 컴포넌트 스타일을 디자인 토큰 기반으로 정리
- 설정 메뉴에서 테마 선택 가능
- 테마 변경 시 주요 화면의 가독성, 대비, 로고/이미지 표시 상태 확인
- 향후 사용자 커스텀 테마 확장 가능성 검토

## 선행 조건
- 모바일/태블릿/PC 화면비 개발 기준 정의 이슈와 함께 진행하거나 그 이후 진행 권장

## 완료 기준
- [ ] 테마 토큰 구조 정의
- [ ] 설정 메뉴에 테마 선택 옵션 추가
- [ ] 핵심 화면에서 다크/라이트 모드 시각 검증
- [ ] 기존 디자인이 깨지지 않도록 회귀 테스트 또는 스크린샷 검증
