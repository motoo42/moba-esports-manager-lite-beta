## 문제
데이터 저장 메뉴의 하위 메뉴 불러오기, 자동 저장은 별도 하위 메뉴로 둘 필요가 없다.

## 기대 동작
- 좌측 데이터 저장 메뉴에는 하위 메뉴가 나타나지 않는다.
- 데이터 저장 클릭 시 현재처럼 저장 화면 하나에서 저장/불러오기/자동 저장 상태를 모두 관리한다.

## 완료 기준
- 데이터 저장 하위 메뉴가 제거된다.
- /saves 화면 기능은 유지된다.
- 라우팅 테스트가 수정된다.
