## 배경

스토브리그 생략 후 바로 시작 옵션을 사용했을 때, 특정 팀에 소속되어 있는 일부 선수가 FA 명단에서 FA 선수처럼 노출되는 현상이 확인되었습니다.

현재 관찰된 재현 경로는 `스토브리그 생략 후 바로 시작`이지만, 다른 진입 경로에서도 같은 데이터 불일치가 발생하는지 확인이 필요합니다.

## 요구사항

- 스토브리그 생략 후 바로 시작 옵션에서 FA 명단이 어떻게 구성되는지 확인합니다.
- `currentTeam`, FA 풀 등록 목록, 화면 필터 조건이 서로 불일치하는 지점을 찾습니다.
- 일반 스토브리그 진입, 시즌 전환, 저장/불러오기 이후에도 같은 문제가 발생하는지 확인합니다.
- 팀 소속 선수가 FA 명단에 노출되지 않도록 데이터 정규화 또는 필터 조건을 수정합니다.
- 이미 소속팀이 있는 선수와 진짜 FA 선수를 구분하는 테스트를 추가합니다.

## 완료 기준

- 팀에 소속된 선수는 FA 명단에 노출되지 않습니다.
- 스토브리그 생략 후 바로 시작 옵션에서도 FA 명단이 정상적으로 표시됩니다.
- 일반 스토브리그 진행 경로에서도 같은 문제가 재발하지 않습니다.
- 관련 회귀 테스트가 추가됩니다.

## 확인 필요

- 현재 문제 선수가 특정 로스터 티어, 특정 팀, 특정 데이터 소스에만 집중되어 있는지 확인해야 합니다.
- 화면 표시 문제인지, 도메인 FA 풀 데이터 문제인지 분리해서 확인해야 합니다.
