## Body

﻿## 문제
현재 로스터 관리 메뉴의 하위 메뉴 순서가 플레이 흐름과 맞지 않는다.

## 요구사항
- 좌측 `로스터 관리` 하위 메뉴 순서를 `선발 5인`, `2군`, `계약` 순서로 변경한다.
- 실제 URL은 기존 `/roster/main`, `/roster/academy`, `/roster/contracts`를 유지한다.
- 화면 내부 기능은 변경하지 않고 메뉴 순서만 조정한다.

## 완료 기준
- 좌측 로스터 관리 하위 메뉴가 `선발 5인 / 2군 / 계약` 순서로 표시된다.
- 각 하위 메뉴 클릭 시 기존 해당 화면으로 정상 이동한다.
- 관련 라우팅/통합 테스트가 유지된다.
