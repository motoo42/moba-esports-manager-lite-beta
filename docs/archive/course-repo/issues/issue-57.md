## 배경

#52에서 스토브리그 최초 진입 룰 안내가 추가되면서, 플레이어가 처음 접하는 복잡한 메뉴나 탭에는 짧은 가이드가 큰 도움이 된다는 점이 확인되었습니다.

장기적으로는 스토브리그뿐 아니라 로스터, 경기 준비, 대회 현황, 메시지함, 저장/설정 같은 주요 메뉴에도 최초 진입 가이드를 확장하면 플레이 흐름을 더 자연스럽게 안내할 수 있습니다.

## 요구사항

- 주요 메뉴/탭별 최초 진입 가이드 시스템을 장기적으로 검토합니다.
- 가이드는 플레이어가 바로 이해할 수 있도록 짧고 상황 중심으로 작성합니다.
- 메뉴별 가이드는 반복 노출을 막고, 필요할 때 다시 볼 수 있어야 합니다.
- 설정 메뉴에서 초기 가이드 제공 여부를 ON/OFF할 수 있도록 확장합니다.
- 가이드 읽음 상태는 커리어별 상태와 전역 설정을 구분해 저장합니다.
- 현재 스토브리그 가이드 구조를 기반으로 재사용 가능한 가이드 모델/컴포넌트로 확장할 수 있는지 검토합니다.

## 완료 기준

- 주요 메뉴/탭별 가이드 후보 목록이 정리됩니다.
- 최초 진입 자동 안내, 다시 보기, 반복 노출 방지 기준이 정의됩니다.
- 설정 메뉴에서 초기 가이드 제공 여부를 관리할 수 있습니다.
- 기존 스토브리그 가이드와 충돌하지 않는 확장 구조가 마련됩니다.

## 범위 제외

- 모든 메뉴의 가이드를 한 번에 구현하는 것은 이번 이슈의 필수 범위가 아닙니다.
- 긴 튜토리얼 캠페인이나 단계별 강제 튜토리얼은 별도 검토 대상입니다.
- 외부 계정 기반 설정 동기화는 이번 범위에 포함하지 않습니다.
