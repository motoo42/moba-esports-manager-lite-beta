## 개요
현재 다음날 진행 대기 시간이 길게 느껴지므로, 기본 진행 시간을 3-4초 수준으로 줄인다.

## 주요 요구사항
- 일반 다음날 진행 오버레이 시간을 3-4초로 조정
- 경기 진행/경기 후 정리/스토브리그 진행 등 상태별 체감 시간 점검
- 스크림 진행 버튼의 대기 시간은 다음날 버튼보다 짧게 유지
- 너무 즉시 넘어가서 진행감이 사라지지 않도록 최소 연출 시간 유지

## 완료 기준
- [ ] 진행 오버레이 최소 대기 시간 조정
- [ ] 경기/비경기/스토브리그 진행에서 정상 동작 확인
- [ ] 기존 테스트 통과
