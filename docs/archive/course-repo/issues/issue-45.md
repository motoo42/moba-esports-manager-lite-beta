﻿## 배경
LCK 구단 상세 화면에 각 팀의 역사와 전통을 보여주는 정보가 부족하다. 구단의 창단 기록, 이름 변경 연혁, 국내/국제대회 우승 경력은 팀 선택과 몰입감에 중요한 정보다.

## 요구사항
- LCK 10개 팀의 실제 역사 데이터를 수집한다.
  - 창단/합류 시점
  - 팀명 변경 연혁
  - 국내대회 우승 경력
  - 국제대회 우승 경력
- 팀 상세 화면에 `[역사]` 섹션을 추가한다.
- 현재 상세 화면의 `[최근 대회]` 항목은 제거한다.
- 데이터 출처를 코드 또는 문서에 남긴다.
- 긴 설명보다 게임 내에서 읽기 좋은 짧은 요약 중심으로 구성한다.

## 완료 기준
- `/teams/:teamId` 상세 화면에서 각 팀의 역사 섹션을 볼 수 있다.
- 최근 대회 섹션은 제거된다.
- 팀별 역사 정보가 실제 데이터 기반으로 작성된다.
- 팀 소개 문구와 역사 섹션의 역할이 구분된다.
