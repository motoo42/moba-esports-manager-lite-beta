## Body

## 배경

게임에 처음 들어온 유저는 각 메뉴가 어떤 역할을 하는지 알기 어렵습니다. 팀을 선택하기 전에 기초 가이드를 제공하면 플레이어가 전체 게임 구조와 주요 메뉴의 의미를 더 쉽게 이해할 수 있습니다.

## 요구사항

- 팀 선택 화면 진입 시 게임 기초 가이드를 제공할 수 있게 합니다.
- 기초 가이드는 게임 내 메뉴와 주요 기능을 설명합니다.
- 여러 장의 화면을 좌우로 넘길 수 있는 모달/슬라이드 형태로 구성합니다.
- 우측 상단에 X 버튼을 넣어 끄고 싶은 사람은 바로 닫을 수 있게 합니다.
- 팀 선택 화면에는 작게 [게임 가이드] 버튼을 유지합니다.
- 저장된 데이터가 없는 경우, 유저가 처음 게임을 하는 것으로 판단해 가이드 창을 자동으로 띄웁니다.
- 저장된 데이터가 있는 경우, 자동으로 띄우지 않고 [게임 가이드] 버튼을 통해 원하는 사람만 볼 수 있게 합니다.
- 데이터 저장 상태를 통해 이 사람이 게임을 해본 적이 있는지 판단합니다.
- 인게임 진입 후에도 [설정] 메뉴에서 동일한 가이드 화면을 다시 확인할 수 있게 합니다.
- 설정 메뉴에서도 [게임 가이드] 버튼을 누르면 X 버튼으로 닫을 수 있는 가이드 창이 열립니다.

## 가이드에 포함할 내용 후보

- 홈 화면의 역할
- 메시지함의 역할
- 로스터 관리의 역할
- 전략/훈련의 역할
- 대회 현황의 역할
- 시즌 캘린더의 역할
- 스토브리그의 역할
- LCK 구단 정보의 역할
- 데이터 저장의 역할
- 시즌 결산과 설정 메뉴의 역할

## 완료 기준

- 신규 유저에게 팀 선택 전에 기초 가이드가 자동 노출됩니다.
- 기존 저장 데이터가 있는 유저에게는 자동 노출되지 않습니다.
- 팀 선택 화면과 설정 화면에서 [게임 가이드] 버튼으로 가이드를 다시 열 수 있습니다.
- 가이드 모달은 좌우 이동과 닫기 동작이 자연스럽게 작동합니다.
- 모바일/데스크톱에서 내용이 잘리거나 겹치지 않습니다.
