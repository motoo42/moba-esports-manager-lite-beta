## 개요
전략/훈련 탭의 상대 리포트와 주간 계획 메뉴가 화면을 더 넓고 풍부하게 활용하도록 개선한다.

## 주요 요구사항
- 상대 리포트 화면이 넓어진 영역을 충분히 활용하도록 레이아웃 재정리
- 주간 계획 화면도 빈 공간을 줄이고 정보 밀도 개선
- 필요하다면 상대 강점/약점, 추천 전략, 선수 상태, 스크림 일정 등 내용을 더 풍부하게 추가
- 가로로 지나치게 긴 단일 박스가 생기지 않도록 카드/패널 구조 조정

## 완료 기준
- [ ] 상대 리포트 화면 레이아웃 개선
- [ ] 주간 계획 화면 레이아웃 개선
- [ ] 데스크톱에서 빈 공간이 과하게 남지 않음
- [ ] 모바일/좁은 화면에서 겹침 또는 잘림 없음
