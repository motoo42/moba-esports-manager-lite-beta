## Body

## 배경

현재 대회 현황 메뉴는 여러 대회의 정보를 한 번에 보여주려다 보니 화면이 복잡하고, 현재 진행 중인 대회로 빠르게 진입한다는 메뉴의 목적이 흐려져 있습니다. 전체 대회 정보를 확인하는 역할은 시즌 캘린더가 담당하고, 대회 현황 메뉴는 현재 진행 중인 대회를 중심으로 정리하는 편이 자연스럽습니다.

또한 LCK Cup 화면은 바론그룹/장로그룹 구조를 충분히 반영하지 못하고 있으며, 대회 화면 전반에서 세로 스크롤이 길어지는 문제가 있습니다.

## 요구사항

### 대회 현황 내비게이션

- `대회 현황` 메뉴를 클릭하면 현재 진행 중인 대회 화면으로 바로 이동합니다.
- 현재 화면 안에 `[시즌 캘린더]` 바로가기 버튼을 추가합니다.
- 사용자가 다른 대회 정보를 보고 싶을 때는 시즌 캘린더를 통해 이동할 수 있게 합니다.
- 대회 현황 메뉴는 “모든 대회 정보 목록”이 아니라 “현재 대회 바로가기”에 가깝게 정리합니다.

### LCK Cup 화면 정리

- LCK Cup 순위표는 10개 팀 통합 순위표가 아니라 바론그룹/장로그룹 순위표로 분리합니다.
- 바론그룹 5팀, 장로그룹 5팀의 순위표가 각각 표시되어야 합니다.
- LCK Cup 화면 우측의 `GROUP` 영역에서 팀 로고와 소속 그룹이 명확하게 보이도록 수정합니다.
- 현재 표시되는 `pts/diff` 정보는 제거하고, 해당 위치에는 `points`만 표시합니다.
- LCK Cup도 다른 대회처럼 `순위`, `일정`, `토너먼트` 등 화면 구분을 갖도록 정리합니다.

### 대회 화면 레이아웃

- 대회 관련 화면에서 전체 페이지 세로 스크롤을 최대한 줄입니다.
- 순위표, 일정, 토너먼트, 브래킷, 대회 소개 등은 한 화면에 모두 쌓기보다 탭/하위 화면으로 분리합니다.
- 현재의 순위표/일정/토너먼트/브래킷 중심 포맷은 유지하되, 대회 소개처럼 우선순위가 낮은 정보는 별도 화면이나 보조 영역으로 분리합니다.
- 전체 레이아웃을 전면 재작성하기보다, 기존 화면 구조 안에서 정보 밀도와 이동 흐름을 정리합니다.

## 완료 기준

- `대회 현황` 메뉴에서 현재 진행 중인 대회로 바로 이동할 수 있습니다.
- 현재 대회 화면에서 시즌 캘린더로 이동하는 버튼이 보입니다.
- LCK Cup은 바론그룹/장로그룹 순위표를 각각 보여줍니다.
- LCK Cup의 그룹 영역에서 팀 로고와 그룹 소속이 명확하게 보입니다.
- LCK Cup 화면이 순위/일정/토너먼트 등 역할별 화면으로 나뉩니다.
- 대회 화면의 불필요한 전체 페이지 세로 스크롤이 줄어듭니다.
- 기존 대회 상세 정보 접근 흐름은 유지됩니다.

## 범위 제외

- 경기 엔진, 대회 결과 계산 로직, 시즌 생성 로직 변경은 이번 이슈 범위가 아닙니다.
- 모든 대회 포맷을 실제 LCK/국제전 규칙과 완전히 일치시키는 작업은 별도 이슈에서 다룹니다.
- 전략/훈련 메뉴의 하위 메뉴 재구성은 이번 이슈 범위가 아닙니다.
