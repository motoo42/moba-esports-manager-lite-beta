## 배경

설정 메뉴는 장기적으로 플레이어가 게임 경험을 조정하는 중심 공간이 되어야 합니다. 현재는 설정 옵션의 구체적인 범위와 저장 방식이 충분히 정의되어 있지 않아, 추후 기능이 늘어날수록 임시 옵션이 흩어질 가능성이 있습니다.

## 요구사항

- 설정 메뉴에 들어갈 옵션 범위를 구체화합니다.
- 옵션별 저장 위치와 적용 시점을 정의합니다.
- 커리어별 설정과 전역 설정을 구분합니다.
- 우선 검토할 설정 후보는 다음과 같습니다.
  - 자동 저장 여부/주기
  - 화면 밀도 또는 UI 축약 옵션
  - 경기 진행 속도/결과 표시 방식
  - 튜토리얼/가이드 다시 보기
  - 메시지/뉴스 노출 빈도
  - 접근성 관련 표시 옵션
- 설정 변경이 기존 저장 데이터와 충돌하지 않도록 마이그레이션 기준을 검토합니다.

## 완료 기준

- 설정 메뉴의 1차 옵션 목록이 정리됩니다.
- 각 옵션의 저장 범위와 적용 타이밍이 명확합니다.
- 설정값이 저장/불러오기 흐름에서 유지됩니다.
- 추후 옵션 추가가 가능한 구조로 정리됩니다.

## 범위 제외

- 모든 설정 후보를 한 번에 구현하는 것은 이번 이슈의 필수 범위가 아닙니다.
- 서버 계정 기반 설정 동기화는 별도 검토 대상입니다.
