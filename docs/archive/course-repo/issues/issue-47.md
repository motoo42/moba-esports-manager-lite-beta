## Body

## 배경

현재 대회 현황 화면의 LCK 포맷 설명이 영어 중심으로 남아 있어 플레이 흐름을 이해하기 어렵습니다. 또한 다른 대회를 확인하려면 시즌 캘린더를 우회해서 들어가야 하는데, 대회 현황 메뉴 안에서도 대회 목록을 직접 확인하고 이동할 수 있어야 합니다.

## 수정 방향

- LCK Cup, LCK Rounds 1-2, LCK Rounds 3-4/3-5, 포스트시즌 등 LCK 관련 포맷 설명을 한글로 정리합니다.
- 현재 구현된 포맷 기준으로 참가팀, 진행 방식, 진출권, 핵심 규칙을 한국어로 설명합니다.
- 대회 현황 메뉴에 `목록` 화면을 추가해 현재 시즌의 대회들을 한눈에 볼 수 있게 합니다.
- 시즌 캘린더에서 대회를 클릭해 들어가는 것처럼, 대회 목록에서도 각 대회 상세 화면으로 이동할 수 있게 합니다.
- 현재 `개요 | 일정/결과 | 브래킷` 같은 하위 메뉴는 좌측 메뉴에 굳이 표시하지 않아도 됩니다.
- 화면 내부에 이미 탭/바로가기 버튼이 있으므로 좌측 하위 메뉴는 단순하게 유지합니다.

## 완료 기준

- LCK 포맷 설명이 영어가 아니라 한글 중심으로 표시됩니다.
- 대회 현황에서 대회 목록을 열 수 있습니다.
- 대회 목록에서 LCK Cup, First Stand, LCK Rounds, MSI, Worlds 등 현재 시즌 대회 상세로 이동할 수 있습니다.
- 좌측 메뉴의 대회 하위 메뉴가 과도하게 복잡하지 않습니다.
- 기존 대회 상세 탭/화면 이동은 유지됩니다.

## 범위 제외

- 경기 엔진이나 실제 대회 일정 생성 로직 변경은 이번 이슈 범위가 아닙니다.
- MSI/Worlds 포맷 자체의 세부 재설계는 별도 이슈에서 다룹니다.
