## 문제
메시지함에서 메시지 제목이 길어지면 우측 중요도 표시 배지가 변형된다. 예: (일반) 표시가 깨진다.

## 기대 동작
- 제목이 길어도 중요도 배지는 고정 크기/형태를 유지한다.
- 제목은 말줄임 또는 줄바꿈 정책을 따른다.

## 완료 기준
- 긴 제목 메시지 fixture로 UI 회귀 테스트를 추가한다.
- 중요도 배지가 찌그러지지 않는다.
