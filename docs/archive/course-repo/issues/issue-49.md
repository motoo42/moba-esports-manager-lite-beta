## Body

## 배경

일부 선수 카드 레이아웃이 화면 안에서 안정적으로 유지되지 않고, 특정 목록에서는 선수 상세 카드를 열 수 없는 문제가 있습니다. 로스터 관리와 선수 확인 흐름에서 자주 마주치는 UI이므로 우선 수정이 필요합니다.

## 요구사항

- 1군 후보 카드가 가로로 과도하게 늘어지는 문제를 수정합니다.
- 2군 선수 카드가 영역 밖으로 잘리거나 내용이 가려지는 문제를 수정합니다.
- 카드 크기, 줄바꿈, overflow 처리를 정리해 선수 카드가 부모 영역 안에서 안정적으로 표시되도록 합니다.
- `로스터 관리 > 계약` 하위 메뉴의 선수 리스트에서도 항목 클릭 시 선수 카드 상세 창이 열리도록 합니다.
- 계약 목록에서 열리는 선수 상세 창은 기존 선수 카드 클릭 시 열리는 상세 창과 동일한 정보를 보여줘야 합니다.

## 완료 기준

- 1군 후보 카드가 의도하지 않게 가로로 늘어나지 않습니다.
- 2군 선수 카드의 이름, 포지션, 능력치 등 주요 정보가 잘리지 않습니다.
- 로스터 관리의 `[계약]` 하위 메뉴에서 선수 리스트 항목을 클릭하면 선수 상세 창이 표시됩니다.
- 같은 선수는 어느 화면에서 열어도 동일한 상세 정보를 보여줍니다.
- 카드 레이아웃 수정 후에도 기존 로스터/계약 기능의 동작은 유지됩니다.

## 범위 제외

- 선수 능력치, 계약 금액, 성장 로직 변경은 이번 이슈 범위가 아닙니다.
- 선수 상세 창의 정보 구조를 새로 설계하는 작업은 별도 이슈에서 다룹니다.
