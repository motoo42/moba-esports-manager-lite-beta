## 배경

현재 LCK Cup 순위표 화면 안에서 Baron / Elder 그룹 포인트 영역을 함께 보여주고 있습니다. 하지만 바론 그룹 순위표와 장로 그룹 순위표만으로도 어느 팀이 어느 그룹에 있는지는 충분히 알 수 있으므로, 그룹 포인트는 별도 위치로 분리하는 편이 더 명확합니다.

## 요구사항

- LCK Cup 하위 메뉴에 [그룹 포인트] 버튼을 추가합니다.
- [그룹 포인트] 버튼은 [순위표] 버튼과 [일정] 버튼 사이에 배치합니다.
- [그룹 포인트]를 클릭하면 현재 순위표에서 보던 Baron / Elder 그룹 포인트 영역을 바로 볼 수 있게 합니다.
- 현재 [순위표] 화면에서는 Baron / Elder 그룹 포인트 영역을 제거합니다.
- 즉, 기존 그룹 포인트 영역을 삭제하는 것이 아니라 새로운 하위 메뉴로 이동합니다.
- [순위표] 화면의 바론 그룹/장로 그룹 카드 우측 상단에 표시되는 [5팀] 텍스트를 제거합니다.
- 해당 위치에는 각 그룹의 현재 점수를 표시합니다.
- 점수 표기는 `0Points` 형태로 표시합니다.
- Points의 P는 반드시 대문자로 표기합니다.

## 완료 기준

- LCK Cup에 [그룹 포인트] 하위 메뉴가 추가됩니다.
- 순위표 화면에서 그룹 포인트 영역이 사라지고, 전용 화면에서 확인됩니다.
- 바론/장로 그룹 순위표 우측 상단에 팀 수 대신 그룹 점수가 표시됩니다.
- 기존 LCK Cup 데이터 계산과 충돌하지 않습니다.
