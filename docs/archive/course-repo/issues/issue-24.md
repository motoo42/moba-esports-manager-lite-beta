## 문제

## 기대 동작

- 좌측 메뉴 상태와 메시지함 내부 필터 상태가 어긋나지 않는다.

## 완료 기준
- 중요, 일정, 이적 클릭 시 /inbox 화면에서 해당 필터가 적용된다.
- 직접 URL 또는 내부 상태 방식 중 하나로 일관성 있게 구현된다.
- 통합 테스트가 추가된다.
