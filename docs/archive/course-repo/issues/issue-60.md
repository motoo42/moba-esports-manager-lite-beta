## 배경

#54에서 주요 메뉴의 세로 길이를 줄이는 작업은 어느 정도 반영되었지만, 홈 화면은 별도의 후속 개편이 필요합니다. 특히 이전에 제거했던 선발 5인 로스터 영역은 홈 화면에서 팀 상태를 직관적으로 보여주는 장점이 있으므로, 더 나은 레이아웃으로 복구할 필요가 있습니다.

## 요구사항

- 홈 화면 전반의 정보 구조와 시각적 우선순위를 다시 정리합니다.
- 이전에 제거했던 5인 로스터 영역을 메인 화면에 복구합니다.
- 복구 시 선수 사진이 잘리거나 카드가 과하게 길어지는 문제를 피합니다.
- 매치 허브, 최근 메시지, 상대 분석, 로스터 요약이 서로 경쟁하지 않도록 배치합니다.
- 홈 화면의 첫 화면 밀도는 유지하되, 중요한 팀 상태가 자연스럽게 보이도록 합니다.

## 완료 기준

- 홈 화면에서 선발 5인 로스터를 다시 확인할 수 있습니다.
- 선수 사진/팀 정보/매치 허브가 겹치거나 잘리지 않습니다.
- 데스크톱과 모바일에서 홈 화면의 핵심 정보가 안정적으로 표시됩니다.
- 기존 #54의 세로 길이 축소 방향과 충돌하지 않습니다.

## 범위 제외

- 매치엔진 화면 구현은 별도 이슈에서 다룹니다.
- 로스터 관리 기능 자체의 구조 변경은 이번 이슈의 필수 범위가 아닙니다.
