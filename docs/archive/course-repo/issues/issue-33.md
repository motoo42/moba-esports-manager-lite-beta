## 문제
좌측 메뉴에서 HB, MS 같은 약어는 메뉴 의미를 직관적으로 전달하지 못한다.

## 기대 동작
- 약어 대신 메뉴 의미를 나타내는 이모티콘, 아이콘, 이미지 중 하나를 사용한다.
- 홈, 메시지함, 로스터, 전략/훈련, 대회, 캘린더, 스토브리그, 구단 정보, 저장, 결산, 설정 등을 더 쉽게 구분할 수 있어야 한다.

## 완료 기준
- 좌측 메뉴 약어 표시가 제거된다.
- 메뉴별 시각 아이콘이 적용된다.
- 아이콘 때문에 레이아웃이 흔들리지 않는다.
