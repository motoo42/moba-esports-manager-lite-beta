## 문제
홈 메뉴의 최근 메시지, 다음 일정 하위 메뉴가 클릭되지 않는다.

## 기대 동작
- 최근 메시지 클릭 시 홈의 최근 메시지 영역으로 이동하거나 메시지함으로 연결된다.
- 다음 일정 클릭 시 홈의 일정 영역 또는 시즌 캘린더로 연결된다.

## 완료 기준
- 홈 하위 메뉴 클릭이 실제 화면 이동/스크롤/탭 전환으로 이어진다.
- 사용자가 현재 어디로 이동했는지 명확히 알 수 있다.
- 관련 라우팅/통합 테스트가 추가된다.
