﻿## 문제
현재 인게임의 여러 팀명이 영문명으로 표시된다. 국내 LCK 중심 게임이므로 일부 화면에서는 한글 팀명 표기가 더 자연스럽다.

## 요구사항
- 인게임에서 팀명을 영문으로 유지할 화면과 한글로 표시할 화면을 구분한다.
- LCK 10개 팀의 한글 표시명을 데이터에 추가한다.
- 좌측 상단 내 팀명, 팀 카드, 구단 정보, 대회 순위표 등 주요 화면에 어떤 표기 정책을 적용할지 정한다.
- 해외팀/국제대회 placeholder는 별도 정책을 둔다.

## 완료 기준
- 팀명 표시 정책 문서 또는 코드 기준이 생긴다.
- LCK 팀은 필요 화면에서 한글명으로 표시된다.
- 기존 teamId, 저장 데이터, 라우팅은 영문/slug 기준으로 안정성을 유지한다.
