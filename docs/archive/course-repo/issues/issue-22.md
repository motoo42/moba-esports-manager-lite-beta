## 문제
이적 로그에서 팀별 이적 로그를 따로 확인할 수 없다.

## 기대 동작
- 이적 로그 화면에 팀별 하위 항목 또는 필터가 존재한다.
- 우리 팀 로그, 전체 로그, 특정 팀 로그를 구분해 볼 수 있다.
- 우리 팀 관련 로그 하이라이트는 유지한다.

## 완료 기준
- 팀별 필터 UI가 추가된다.
- 선택 팀의 이적 로그만 볼 수 있다.
- 닫힌 시장 정보 화면과 활성 스토브리그 화면에서 동일한 UX를 제공한다.
