﻿## 배경
현재 LCK 구단 정보에는 `[구단 목록]` 하위 메뉴가 남아 있는데, 단일 기능 메뉴라면 불필요하다. 또한 팀 목록 화면은 시작 팀 선택 화면처럼 10개 팀을 보기 좋게 중앙 배치하는 편이 자연스럽다.

## 요구사항
- 좌측 메뉴의 `LCK 구단 정보 > 구단 목록` 하위 메뉴를 제거한다.
- `LCK 구단 정보` 메인 메뉴 클릭만으로 `/teams`에 진입한다.
- `/teams` 화면은 LCK 10팀을 2x5 카드 형태로 보여준다.
- 카드 그리드는 화면 중앙에 안정적으로 배치한다.
- 팀 소개 문구는 목록 카드에 바로 노출하지 않고, 팀 상세 화면에서만 보여준다.
- 목록 카드는 로고, 팀명, 티어/전력/예상 순위 정도의 핵심 정보만 표시한다.

## 완료 기준
- LCK 구단 정보 메뉴에 불필요한 하위 메뉴가 없다.
- `/teams` 화면에서 10개 팀이 중앙 2x5 형태로 정렬된다.
- 목록 카드에서 긴 소개 문구가 빠져 화면이 깔끔해진다.
- 팀 카드 클릭 시 상세 화면에서 소개 문구를 볼 수 있다.
