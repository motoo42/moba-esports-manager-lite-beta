## Body

﻿## 문제
현재 홈 메뉴의 `대시보드`, `최근 메시지`, `다음 일정` 하위 메뉴는 실제 플레이 흐름에서 의미가 크지 않다.

## 요구사항
- 좌측 `홈` 메뉴의 하위 메뉴를 제거한다.
- `홈` 클릭 시 `/hub` 대시보드 화면으로만 이동한다.
- 메인 허브 내부의 최근 메시지/다음 일정 섹션 자체는 유지한다.

## 완료 기준
- 홈 메뉴를 클릭해도 하위 메뉴가 펼쳐지지 않는다.
- `/hub` 이동은 정상 유지된다.
- 최근 메시지/다음 일정은 메인 허브 본문에서 계속 확인할 수 있다.
