## 배경

현재 팀 로고는 팀마다 흰색 배경 박스가 보입니다. 흰색 배경 자체는 문제 없지만, 일부 팀 로고가 박스 안에서 작게 보이거나 정중앙에 배치되지 않는 문제가 있습니다. 특히 좌측 상단 로고에서 T1 로고가 흰 박스 정중앙에 들어오지 않는 현상을 확인했습니다.

## 요구사항

- 팀 로고가 흰색 박스 안에서 시각적으로 중앙에 오도록 정렬합니다.
- 로고별 크기 차이가 지나치게 크게 느껴지지 않도록 보정합니다.
- 좌측 상단 팀 로고도 동일한 정렬 규칙을 따르게 합니다.
- T1 로고를 포함해 모든 LCK 팀 로고를 점검합니다.
- 흰색 배경 박스 자체는 유지해도 괜찮습니다.

## 완료 기준

- 주요 화면에서 팀 로고가 흰 박스 중앙에 안정적으로 배치됩니다.
- 특정 팀 로고만 지나치게 작거나 치우쳐 보이지 않습니다.
- 좌측 상단 팀 로고와 본문 카드 로고의 정렬 품질이 일관됩니다.
- 데스크톱과 모바일에서 로고가 잘리거나 찌그러지지 않습니다.
