## 개요
전략/훈련 탭의 스크림 메뉴에서 상대 팀을 고를 때 팀 로고를 함께 표시해 가독성과 선택감을 높인다.

## 주요 요구사항
- 스크림 상대 선택 카드 또는 목록에 팀 로고 표시
- 로고 크기, 정렬, 흰색 박스 중앙 정렬 기준을 기존 팀 로고 컴포넌트와 맞춘다.
- 비활성화된 상대도 로고와 disabled 상태가 함께 자연스럽게 보이도록 처리

## 완료 기준
- [ ] 스크림 상대 선택 UI에 팀 로고 표시
- [ ] disabled 상대 표시가 깨지지 않음
- [ ] 모바일/좁은 화면에서 카드가 과하게 늘어나지 않음
