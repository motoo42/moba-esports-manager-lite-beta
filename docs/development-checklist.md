## 0. 현재 진척도 요약

산정 시점: 2026-05-30

### 체크리스트 기준 진척도

- 전체 체크박스: 335개.
- 완료 체크박스: 185개.
- 미완료 체크박스: 150개.
- 단순 체크율: 약 55.2%.

주의: 이 수치는 프로젝트 세팅, 폴더 구조, 테스트 예시처럼 초반 기반 작업도 모두 같은 무게로 계산한 값입니다. 실제 게임 플레이 완성도는 이보다 낮게 봅니다.

### 목표별 구현 완성도 추정

| 목표 기준 | 현재 구현률 | 판단 |
| --- | ---: | --- |
| 핵심 프로토타입 | 94~96% | 커리어 생성, 스토브리그, 로스터/계약 확정, FM식 메인 프레임, 전략/훈련 계획, 선발 교체, 선수 상태 변화, 상대 분석, 밴픽 점수 기반 시리즈 시뮬레이션, LCK Cup 완주 흐름, 일일 진행 엔진, LCK Rounds 1-2 진행 엔진과 대회 현황/포스트시즌 브래킷 UI까지 연결됨 |
| 1시즌 플레이 가능한 MVP | 64~70% | 시즌 상태, 계약, 주간 계획, 선발 교체, 선수 상태 변화, 상대 분석, 메인 UI, LCK Cup 완주 흐름, 실제 달력 기반 LCK Rounds 1-2 일정 생성/진행/완료 감지, 순위표/일정/토너먼트 UI는 생겼으나 포스트시즌 경기 진행/국제전 본구현/시즌 종료가 없음 |
| 정교한 개인 프로젝트 v1 | 19~26% | 구조는 잘 잡혔고 밴픽/계약/주간 계획/UI 기반이 생겼지만 장기 커리어, 저장, 이벤트, 밸런싱, 실제 데이터 확장이 대부분 남음 |
| 장기 플레이 완성형 | 3~5% | 20시즌 안정성, 풍부한 데이터, 고급 이벤트, UI 폴리싱은 아직 초기 단계 |

현재 프로젝트는 "앱 기반 공사, 스토브리그 로스터 확정, FM식 메인 프레임, 주간 계획, 시즌 중 선발 교체, 폼/피로도/사기 상태 변화, 상대 분석, 밴픽 점수 기반 경기 시뮬레이션 초안, LCK Cup 전체 간소화 진행, 매치 허브 날짜별 일정/결과/그룹 현황 표시, LCK Cup 대회 현황 화면, 실제 달력 기반 LCK Rounds 1-2 일정 생성/진행/완료 감지, LCK Rounds 1-2 순위표/일정/토너먼트 대회 현황 UI"까지 완료된 상태입니다. 사용자가 오래 플레이할 수 있는 게임으로 보려면 아직 이후 국제전, 포스트시즌 경기 진행, 시즌 종료, 저장, 장기 시즌 전환, 밸런싱이 많이 남아 있습니다.

### 남은 개발 시간 추정

| 목표 | 추가 예상 시간 | 설명 |
| --- | ---: | --- |
| 핵심 프로토타입 완성 | 2~6시간 | LCK Rounds 1-2 포스트시즌 경기 진행까지 확인 가능한 수준 |
| 1시즌 플레이 가능한 MVP | 22~44시간 | 포스트시즌, 국제전 축약 진행, 시즌 요약, 저장 일부 포함 |
| 정교한 개인 프로젝트 v1 | 168~276시간 | 20시즌 구조, 실제 데이터 확장, 이벤트, 장기 커리어, 저장, UI 정리 포함 |
| 장기 플레이 완성형 | 360~580시간 | 아시안게임, 국제전 깊이, 밸런싱 반복, 커스텀 테마, 장기 테스트 포함 |

현재 목표가 제출형이 아니라 정교한 개인 프로젝트라면, 현실적인 다음 큰 완성 지점은 "정교한 개인 프로젝트 v1"이며 남은 시간은 약 158~262시간으로 봅니다.
