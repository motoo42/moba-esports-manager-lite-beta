# MVP 범위와 화면 흐름

## 1. MVP 목표

1시즌을 처음부터 끝까지 플레이할 수 있는 e스포츠 매니저 게임

MVP의 최종 기준:

- 커리어 시작
- 스토브리그
- 로스터 확정
- LCK Cup
- First Stand
- LCK Rounds 1-2
- MSI
- LCK 후반 시즌
- Worlds
- 시즌 요약
- 다음 시즌 진입

현재는 LCK Cup과 LCK Rounds 1-2의 핵심 흐름이 구현된 상태

## 2. MVP에 포함

### 게임 진행

- 커리어 생성
- 팀명 입력
- 2026 시즌 시작
- 스토브리그 후 LCK Cup 활성화
- 날짜 단위 진행
- 우리 팀 경기일에는 플레이 버튼 표시
- 경기 후 리뷰 상태 표시
- 진행중 오버레이 표시

### 로스터

- LCK 선수 기반 로스터 구성
- 팀당 1군+2군 10~15명
- 선발 5인 관리
- 같은 포지션 드래그 앤 드롭 교체
- 계약 타입: 1년, 2년, 1+1년
- 폼, 피로도, 사기
- 선수 상세 스탯 팝업

### 전략/훈련

- 6개 전략 스타일
- 4개 훈련 강도
- 전략 상성
- 상대 분석
- 경기 전망 등급
- 밴픽 점수 기반 경기력 보정

### 대회

- LCK Cup
- First Stand
- LCK Rounds 1-2
- MSI
- LCK Rounds 3-4 또는 3-5
- Asian Games
- Worlds

### 화면

- 메인 허브
- 로스터 관리
- 전략/훈련
- 대회 현황
- 대회별 상세 URL
- 시즌 캘린더
- 시즌 요약

### 테스트

- 단위 테스트
- 통합 테스트
- 시스템 테스트
- 인수 테스트
- 16:9 스크린샷 검증

## 3. MVP에서 제외하거나 후순위

- 실제 Riot API 연동
- 실시간 e스포츠 데이터
- 로그인/계정 시스템
- 멀티플레이
- 온라인 랭킹
- 실제 선수 사진 파일 저장
- 실제 팀 로고 파일 저장
- 복잡한 이적 시장
- 바이아웃
- 트레이드
- 임대
- 고급 스폰서 시스템
- 고급 재정 시스템
- 완전한 챔피언 밴픽 UI
- 실제 경기 뷰어
- 장기 20시즌 밸런싱 완료
- 커스텀 테마 기능

## 4. 현재 화면 흐름

```text
/
커리어 시작
  ↓
/roster
스토브리그 로스터 확정
  ↓
/hub
메인 허브
  ├─ /roster
  ├─ /match
  ├─ /competitions
  ├─ /competitions/:id
  ├─ /calendar
  └─ /summary
```

## 5. 메인 루프

```text
오늘 확인
-> 로스터/전략/일정 점검
-> 진행 또는 플레이
-> 경기 결과 반영
-> 선수 상태와 순위표 갱신
-> 다음 날짜
```

## 6. 1시즌 MVP 완료 기준

완료:

남음:

- LCK 후반 시즌 완주

- 3시즌 연속 플레이 안정성

- MongoDB 기반 커리어 저장
- 풍부한 이벤트/뉴스 카드
- 밴픽과 메타 변화 고도화
- UI 폴리싱
