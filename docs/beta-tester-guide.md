# MOBA Esports Manager Lite 베타 테스트 안내

이 문서는 베타테스터가 링크 하나로 접속해 게임을 체험하고 피드백을 남기기 위한 안내서다.

## 접속 링크

```text
https://moba-esports-manager-lite-beta.onrender.com/
```

처음 접속할 때는 서버가 깨어나는 데 시간이 걸릴 수 있다. 화면이 바로 뜨지 않으면 30초 정도 기다린 뒤 새로고침한다.

## 권장 환경

권장:

- PC 또는 노트북
- Chrome 또는 Edge
- 일반 브라우저 창
- 1280x720 이상 가로 화면

권장하지 않음:

- 모바일 세로 화면
- 시크릿 모드
- 여러 사람이 같은 PC/같은 브라우저 프로필을 공유하는 방식

현재 모바일 세로 화면은 지원하지 않는다. 화면이 너무 작으면 PC 또는 큰 가로 화면으로 접속하라는 안내가 표시된다.

## 저장 데이터 주의

현재 베타는 로그인 계정이 없다. 대신 브라우저마다 자동으로 저장 공간을 나눈다.

```text
내 Chrome -> 내 저장 목록
친구 Chrome -> 친구 저장 목록
```

보통 각자 자기 PC/브라우저로 접속하면 저장이 섞이지 않는다.

주의할 상황:

- 같은 PC의 같은 브라우저를 같이 쓰면 저장 목록이 같이 보일 수 있다.
- 시크릿 모드는 창을 닫으면 저장 식별 정보가 사라질 수 있다.
- 브라우저 데이터나 localStorage를 삭제하면 저장 목록이 새로 잡힐 수 있다.
- Chrome에서 만든 저장은 Edge에서 바로 보이지 않을 수 있다.

저장명을 만들 때는 누가 만든 저장인지 알 수 있게 적어주면 좋다.

```text
이름-선택팀-날짜
예: 민수-T1-0609
```

## 테스트 목표

이번 베타의 목적은 완성된 게임 평가가 아니라, 실제 플레이어가 처음 만졌을 때 어디서 막히고 무엇이 헷갈리는지 찾는 것이다.

중점적으로 봐줬으면 하는 것:

- 처음 들어왔을 때 무엇을 해야 하는지 알 수 있는가
- 좌측 메뉴 이름과 화면 이동이 이해되는가
- 진행 버튼을 누르며 시즌이 흘러가는 느낌이 있는가
- 로스터, 전략/훈련, 대회 현황, 시즌 캘린더가 서로 연결되어 보이는가
- 저장/불러오기 위치를 찾을 수 있는가
- 글자가 잘리거나 버튼이 겹치는 화면이 있는가
- 게임이 멈추거나 흰 화면이 뜨는 구간이 있는가

## 10분 빠른 테스트

처음 만져볼 때 추천하는 흐름이다.

1. 접속 링크로 들어간다.
2. 커리어를 시작한다.
3. 팀명/팀 설정 화면을 지나 프리시즌 스토브리그로 들어간다.
4. 재계약 또는 방출 화면을 훑어본다.
5. 좌측 메뉴에서 `로스터 관리`, `전략 / 훈련`, `대회 현황`, `시즌 캘린더`, `데이터 저장`을 한 번씩 눌러본다.
6. `데이터 저장` 화면에서 수동 저장을 해본다.
7. 새로고침 후 저장을 불러올 수 있는지 확인한다.

이 흐름에서 헷갈리는 지점이 있으면 가장 중요하게 알려주면 된다.

## 30분 기본 플레이

조금 더 오래 테스트할 수 있다면 아래 흐름을 추천한다.

1. 프리시즌 스토브리그 1주차에서 선수 재계약/방출을 진행한다.
2. 하루 진행 버튼을 눌러 날짜가 넘어가는지 확인한다.
3. 2주차 이후 FA 시장에서 선수를 둘러본다.
4. 로스터 화면에서 1군/2군, 계약 현황을 확인한다.
5. 전략과 훈련 강도를 바꿔본다.
6. LCK Cup 또는 다음 대회까지 진행해본다.
7. 경기 결과와 선수 상태 변화가 납득되는지 본다.
8. 대회 현황과 시즌 캘린더에서 현재 시즌 위치를 확인한다.
9. 다시 저장하고 불러온다.

## 길게 테스트할 때

가능하면 자동 진행을 많이 눌러 아래 지점까지 도달해본다.

- LCK Cup
- First Stand
- MSI
- Asian Games 선택
- Worlds
- 시즌 요약
- 스토브리그
- 다음 시즌 진입

중간에 진행이 막히거나 무엇을 해야 하는지 모르겠는 날이 있으면 알려주면 좋다.

## 버그 제보 방법

버그를 발견하면 아래 형식으로 알려주면 가장 좋다.

```text
화면:
내가 누른 것:
예상한 결과:
실제로 나온 결과:
다시 해도 반복되는지:
스크린샷 있음/없음:
```

예시:

```text
화면: 스토브리그 FA 시장
내가 누른 것: 선수 협상 버튼
예상한 결과: 계약 제안 창이 열림
실제로 나온 결과: 화면이 멈춤
다시 해도 반복되는지: 2번 반복됨
스크린샷 있음/없음: 있음
```

스크린샷은 가능하면 화면 전체가 보이게 찍어주면 좋다.

```text
도달한 지점:

첫 화면 이해도 1~5:
메뉴 이해도 1~5:
로스터 화면 이해도 1~5:
스토브리그 이해도 1~5:
경기 진행 재미 1~5:
대회 현황 이해도 1~5:

전체 UI 가독성 1~5:
다시 해보고 싶은 정도 1~5:

헷갈렸던 점:

가장 먼저 고쳤으면 하는 것:

```

## 현재 알고 있는 미완성/수정 예정 항목

아래는 이미 인지하고 있는 부분이다. 그래도 실제로 얼마나 불편한지 느낀 점을 알려주면 도움이 된다.

- 커리어 시작 시 LCK 10개 팀 중 하나를 고르는 흐름으로 더 명확하게 바꿀 예정.
- 팀 로고, LCK 로고 등 공식적인 시각 asset은 아직 충분히 적용되지 않았다.
- 1군/2군 연봉과 팀 예산 밸런스는 조정 예정이다.
- 로그인 계정은 아직 없고, 저장은 브라우저 기준으로 분리된다.
- 모바일 세로 화면은 지원하지 않는다.
- 선수 사진은 베타용 임시 asset이며 일부는 최신 사진이 아닐 수 있다.
- 메시지함/뉴스/일정 알림은 아직 1차 구현 전이다.
- 타팀 로스터/스카우팅 화면은 아직 부족하다.
