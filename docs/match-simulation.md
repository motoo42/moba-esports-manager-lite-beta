# 경기 시뮬레이션

## 1. 기본 방향

경기 결과는 팀 전력, 선수 상태, 전략/훈련, 밴픽 점수, 랜덤 변동값을 합산해 계산

사용자의 주요 개입 지점:

- 선발 5인
- 전략 스타일
- 훈련 강도
- 상대 분석
- 밴픽 방향

## 2. 현재 구현 요소

- seeded random 기반 결과 재현
- 단판 경기 시뮬레이션
- BO3/BO5 시리즈 시뮬레이션
- 팀 전력 계산
- 전략 보너스
- 훈련 강도 보너스
- 선수 폼/피로도/사기 보정
- 팀 케미 보정
- 밴픽 점수 보정
- 경기 결과 로그
- MatchRecord 저장

## 3. 현재 계산 흐름

```text
선발 5인 평균 전력
+ 선수 세부 스탯
+ 전략 보너스
+ 훈련 강도 보너스
+ 선수 상태 보정
+ 팀 케미 보정
+ 밴픽 점수
+ 랜덤 변동값
= 팀 전력
```

승률 계산은 ELO식 확률 모델을 기반으로 확장 가능

```text
winProbability = 1 / (1 + 10 ^ ((opponentPower - teamPower) / 400))
```

## 4. 전략 스타일

현재 6개 전략:

- 공격 지향
- 템포 지향
- 운영 지향
- 시야 중심형
- 후반 설계형
- 균형 전술

전략은 상대 스타일과 상성 관계를 가짐

UI에는 정확한 숫자 대신 다음 정보를 표시:

- 유리한 스타일
- 불리한 스타일
- 경기 전망 등급

## 5. 훈련 강도

현재 4개 강도:

- 고강도 훈련
- 일반 훈련
- 가벼운 훈련
- 휴식

훈련 강도는 경기 준비 보너스와 선수 상태 변화에 연결

## 6. 밴픽 점수

1차 구현은 정량 점수 기반

고려 요소:

- 선수 선호 챔피언
- 선수 비선호 챔피언
- 메타픽
- 팀 조합 태그
- 상대 스타일 대응
- 피어리스 시스템 확장 가능성

목표:

- 밴픽 점수를 경기 승률에 반영
- 플레이어가 밴픽 결과를 객관적 등급으로 확인
- 추후 실제 밴픽 UI와 챔피언 풀 확장 가능

## 7. 결과 처리

경기 결과 후 반영:

- 승패 기록
- 세트 스코어
- 순위표 갱신
- 선수 폼 변화
- 선수 피로도 변화
- 선수 사기 변화
- 최근 경기 결과 표시
- 매치 리뷰 표시

## 8. 국제전 보정

국제전 상대는 정적 데이터로 시작

확장 후보:

- 리그별 평균 전력
- 리그별 스타일
- 국제전 적응력
- 장거리 일정 피로
- 낯선 메타
- 패치 적응

## 9. 남은 개선

- 정확한 승률 수치 표시 여부 결정
- 밴픽 UI 구현
- 실제 챔피언 데이터 확장
- 패치/메타 변화
- 국제전 팀별 전력 밸런싱
- 20시즌 장기 시뮬레이션 안정성 검증
