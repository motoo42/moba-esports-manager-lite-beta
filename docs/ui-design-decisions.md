# UI 설계 결정사항

## 1. 기본 방향

- Football Manager식 메인 허브 구조
- 16:9 가로 화면 우선
- PC와 가로 태블릿 기준
- 정보 밀도 높게 구성
- 전체 프레임 고정
- 상단 헤더, 좌측 메뉴, 진행 버튼은 고정
- 스크롤은 전체 페이지가 아니라 중앙 콘텐츠나 패널 내부에서만 발생
- UI 변경 후 16:9 스크린샷으로 확인

## 2. 기준 이미지

사용자 제공 참고 이미지:

- `UI-Picture/` 아래 메인 허브 참고 이미지
- `UI-Picture/` 아래 로스터 참고 이미지
- `UI-Picture/` 아래 로드맵 참고 이미지

현재 스크린샷 저장 위치:

- `test-results/ui-screenshots/`
- `screenshots/live-preview/`

## 3. 공통 프레임

공통 프레임 구성:

- 좌측 아이콘 메뉴
- 좌측 세부 메뉴
- 상단 헤더
- 현재 날짜/시즌/대회 표시
- 우측 상단 진행 버튼
- 중앙 콘텐츠 영역

버튼 상태:

- `진행`: 경기 없는 날, 또는 경기 후 다음 날로 이동
- `플레이`: 우리 팀 경기일
- 진행중: 5초 오버레이 표시

## 4. 좌측 메뉴

현재 메뉴 방향:

- 메인 허브
- 로스터 관리
- 전략/훈련
- 스카우트 또는 상대 분석
- 대회
- 캘린더
- 기록/요약

선택된 메뉴는 테두리와 배경으로 강조

세부 메뉴는 선택된 메뉴에 따라 변경

## 5. 메인 허브

메인 허브 구성:

- 현재 선발 5인
- 현재 날짜와 활성 대회
- 다음 경기
- 경기 직전 프리뷰
- 경기 직후 리뷰
- 최근 결과
- 진행/플레이 버튼

매치 허브 상태:

- 경기 전: 프리뷰
- 경기 직후: 리뷰
- 그 외: 다음 경기 정보

## 6. 로스터 관리

현재 구현:

- 상단 선발 라인업
- 하단 후보 목록
- 후보 목록에는 선발 5인 제외
- 같은 포지션끼리만 드래그 앤 드롭
- 드롭 즉시 반영
- 선발/후보 배지는 카드에 표시하지 않음
- 영역 구분으로 선발과 후보 역할 전달
- 카드에는 닉네임, 포지션, 오버롤, 폼, 피로도, 사기 표시
- 카드 클릭 시 상세 스탯 팝업
- 드래그와 클릭 이벤트 충돌 방지

후순위:

- 선수 사진
- 실제 팀 로고
- 선수 상세 프로필 확장
- 1군/2군 이동 UI 고도화

## 7. 선수 상태 표시

폼:

- 숫자 표시

피로도:

- 숫자 표시
- 낮을수록 좋고 높을수록 나쁨

사기:

- 5단계
- 원형 배지
- 흰색 화살표
- 최상: 12시
- 중상: 1.5시
- 중: 3시
- 중하: 4.5시
- 최하: 6시
- 초록색에서 노란색, 빨간색으로 색 변화

## 8. 전략/훈련 화면

전략:

- 공격 지향
- 템포 지향
- 운영 지향
- 시야 중심형
- 후반 설계형
- 균형 전술

훈련 강도:

- 고강도 훈련
- 일반 훈련
- 가벼운 훈련
- 휴식

UI 원칙:

- 숫자 승률보다 등급 중심
- 매 경기 전략 추천보다 유리한 스타일/불리한 스타일 표시
- 선택은 실제 경기 시뮬레이션에 반영

## 9. 대회 현황

공통:

- 화면 제목은 대회명 직접 표시
- 대회별 탭 구성
- 내 팀은 약하게 하이라이트
- 경기 결과는 승리/패배/중립 색상 구분

LCK Rounds 1-2:

- 순위표
- 일정
- 토너먼트
- 상위 6팀 포스트시즌 진출 설명
- 산술적으로 확정된 팀만 PO 배지
- 정규시즌 중 토너먼트 탭은 빈 슬롯 표시
- 포스트시즌 진행 중 현재 라운드 강조

First Stand:

- 개요
- 조별리그
- 일정
- 토너먼트
- 현재는 화면 골격 단계

## 10. 시즌 캘린더

두 가지 보기:

- 로드맵
- 달력

로드맵:

- 국내전/국제전 트랙 분리
- 대회명과 작은 라벨 중심
- 기간만 간결하게 표시
- 대회가 없는 기간은 비워둠
- 대회 클릭 시 상세 화면 이동

달력:

- 실제 양력 달력
- 우리 팀 경기만 표시
- 다른 팀 경기는 대회 일정 화면에서 확인

## 11. 로딩/진행 연출

진행 버튼 클릭 후 즉시 결과를 보여주지 않음

현재 기본값:

- 약 5초 진행중 오버레이
- 하루 진행중
- 오늘의 경기 계산 중
- 선수 상태 반영 중
- 순위표 갱신 중

목적:

- 플레이어에게 하루 단위 분절감 제공
- 추후 무거운 계산, 뉴스, 이벤트 생성에 사용 가능

## 12. 검증 기준

UI 변경 후 확인:

- 16:9에서 프레임이 깨지지 않는지
- 전체 페이지 스크롤이 생기지 않는지
- 패널 내부 스크롤만 작동하는지
- 텍스트가 카드 밖으로 넘치지 않는지
- 메뉴와 진행 버튼이 고정되는지
- 우리 팀 하이라이트가 과하지 않은지
- 색상 의미가 분명한지
