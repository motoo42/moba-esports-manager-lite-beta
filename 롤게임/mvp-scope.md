# LoL Esports Manager MVP 범위와 화면 흐름

## 1. MVP에 포함할 기능

- 실제 LoL Esports 선수명을 사용하는 LCK 선수 풀.
- 기본 지역은 LCK.
- 로스터 구성은 LCK 선수 풀 안에서만 가능.
- 국제 대회에만 등장하는 사전 정의 LPL/LEC/LTA/LCP 팀/선수 데이터.
- 5개 포지션: Top, Jungle, Mid, Bot, Support.
- 선수 능력치.
- FM/FIFA식 현재 능력치와 성장 가능성 체계.
- `어빌`, `포텐`, `종합점수`는 모두 0-100 스케일.
- 예산.
- 로스터 선택.
- 계약 기간: 1년, 2년, 1+1년 옵션.
- 전략 설정.
- 훈련 설정.
- 선수 교체.
- 상대 분석.
- 팀 케미 계산.
- ELO 기반 경기 시뮬레이션.
- 순위표.
- 뉴스/이벤트 카드.
- 시즌 결과 화면.
- 최대 20시즌 플레이 가능한 커리어 구조.
- 선수 나이 증가, 은퇴, 군 입대 이슈.
- 유망주 자동 생성.
- 기본 LoL/LCK 테마.
- 현실 LoL Esports 시즌을 반영한 LCK Split/국제전 시즌 구조.
- 아시안게임이 있는 해와 없는 해를 구분하는 시즌 캘린더.
- 아시안게임은 4년에 한 번 열리는 중요 이벤트로 처리.

## 2. 추천 시즌 길이

```text
최종 목표: 로스터 구성과 플레이를 합쳐 한 시즌 20시간 수준의 깊이
간소화 목표: 한 시즌 2-3시간 수준
MVP 목표: 핵심 루프를 검증할 수 있는 압축 시즌
```

## 3. 추천 선수 수

```text
LCK 선수 50-70명, LPL/LEC/LTA/LCP 선수는 국제전 상대용으로 별도 정의
```

## 4. 제외할 범위

MVP에서는 제외하거나 후순위로 둡니다.

- 실시간 e스포츠 데이터.
- Riot API 의존성.
- 멀티플레이.
- 온라인 저장 계정.
- 완전한 이적 시장.
- 복잡한 스폰서 시스템.
- 상세 챔피언 밴픽 시뮬레이터.
- 실시간 경기 뷰어.
- 사용자가 LPL/LEC/LTA/LCP 선수를 영입하는 시스템.
- 모든 타 리그 정규 시즌 시뮬레이션.
- 실제 국제 대회 포맷의 완전 재현.
- 한 시즌 20시간 분량의 모든 세부 시스템 완성.
- 복잡한 장기 계약/이적료/바이아웃/트레이드 시스템.
- 공개 배포용 실제 팀/선수 스킨 파일 정리.
- 공개 배포용 실제 팀 로고/선수 사진 포함 여부 검토.
- 아시안게임의 실제 국가대표 선발 규정 완전 재현.

## 5. 주요 사용자 흐름

```text
1. 사용자가 새 시즌을 시작한다.
2. 고정된 팀 예산을 받는다.
3. LCK 선수 풀 안에서 계약 조건을 고려해 로스터를 구성한다.
4. 앱이 팀 전력과 케미를 계산한다.
5. 사용자가 전략, 훈련, 선수 교체, 상대 분석을 설정한다.
6. LCK Split을 진행하며 순위를 올린다.
7. 성적 조건을 만족하면 국제 이벤트에 진출한다.
8. 국제전에서는 사전 정의된 LPL/LEC/LTA/LCP 팀과 선수들이 상대 팀으로 등장한다.
9. 이벤트 카드가 사기, 폼, 예산, 선수 상태 등을 변화시킨다.
10. 시즌 종료 후 나이, 계약, 은퇴, 군 입대, 유망주 생성이 처리된다.
11. 최대 20시즌까지 커리어를 이어간다.
```

## 6. 화면 초안

### 6.1 시작 / 팀 정체성 화면

내용:

- 팀 이름 입력.
- 난이도 선택.
- 기본 지역: LCK.
- 커리어 목표: 최대 20시즌.
- 간단한 규칙 설명.

### 6.2 로스터 빌더

레이아웃:

- 포지션 슬롯.
- 선수 카드.
- 예산 패널.
- 팀 케미 미리보기.

선수 카드 필드:

- 이름.
- 포지션.
- 비용.
- 어빌.
- 포텐.
- 종합점수.
- 피지컬/메카닉.
- 운영.
- 한타.
- 라인전.
- 멘탈.
- 챔피언 폭.
- 특성.

### 6.3 경기 주간 화면

내용:

- 상대 팀 미리보기.
- 전략 옵션.
- 훈련 집중도.
- 선발/교체 선수 선택.
- 상대 분석 카드.
- 팀 준비 상태.
- 시뮬레이션 버튼.
- 결과 로그.

### 6.4 시즌 캘린더 화면

내용:

- LCK Split 1, 2, 3.
- First Stand, MSI, Worlds.
- 아시안게임 연도일 때만 표시되는 특별 이벤트.
- 잠금/진출/진행중/완료 상태.
- 각 이벤트별 진출 조건.
- 시즌 유형: 일반 연도 / 아시안게임 연도.

### 6.5 순위표 / 시즌 요약

내용:

- 승패 기록.
- 순위.
- MVP 선수.
- 팀 강점/약점.
- 계약 만료 선수.
- 은퇴/군 입대 후보.
- 다음 시즌 유망주 풀.
- 다시 시작 버튼.
