# 리스크, AI 활용, 3주 계획

## 1. 리스크

- 밸런스 리스크: 시뮬레이션이 무작위처럼 느껴질 수 있습니다.
- 장기 커리어 리스크: 20시즌 동안 선수 풀, ELO, 예산, 성장/은퇴 시스템이 무너지지 않게 해야 합니다.
- 범위 리스크: 매니저 게임은 기능이 빠르게 커집니다.
- 실제 데이터 유지보수 리스크: LoL Esports 팀/선수/대회명은 시즌마다 바뀔 수 있습니다.
- 콘텐츠 리스크: 실제 선수/팀 데이터가 재미있고 다양해야 합니다.
- 캘린더 리스크: LCK Split/국제전/아시안게임을 모두 넣으면 이벤트 수가 많아질 수 있으므로 각 대회는 압축 진행해야 합니다.
- 커스텀 테마 리스크: 사용자가 넣은 외부 이미지 URL이나 데이터가 깨질 수 있습니다.
- 우선순위 리스크: 커스텀 테마는 재미있는 기능이지만 핵심 게임 루프보다 먼저 구현하면 일정이 밀릴 수 있습니다.

## 2. 개발 과정에서의 AI 활용

AI는 다음에 사용할 수 있습니다.

- 실제 LoL 선수 이름과 특성 정리.
- 실제 LoL 팀 이름 정리.
- LCK 선수 풀과 LPL/LEC/LTA/LCP 국제전 상대 팀 데이터 생성.
- LCK Split/국제전/아시안게임 이벤트 카드 생성.
- FM/FIFA식 선수 능력치 체계 설계.
- 어빌/포텐/종합점수 분포와 밸런스 조정.
- 실제 팀 로고 URL과 색상 테마 정리.
- 이벤트/뉴스 카드 작성.
- 경기 시뮬레이션 공식 설계.
- ELO 기반 승률/결과 계산식 설계.
- 20시즌 장기 커리어 이벤트 설계.
- 신인 생성/은퇴/군 입대 이벤트 데이터 설계.
- 선수 능력치와 비용 밸런싱.
- UI 문구 작성.
- 테스트 시나리오 생성.
- 실제 LoL/LCK/Riot 명칭과 에셋 사용 범위를 개인용으로 유지할지 검토.
- 커스텀 테마 JSON 스키마와 검증 케이스 설계.

MVP에서는 런타임 AI가 필요 없습니다.

## 3. 3주 계획

### 1주차

- 실제 LoL Esports 용어와 데이터 범위 확정.
- 선수 능력치와 시뮬레이션 공식 정의.
- ELO 기반 경기 결과 계산 방식 정의.
- LCK Split 3개, First Stand/MSI/Worlds, 아시안게임을 포함한 압축 시즌 캘린더 정의.
- 일반 연도 캘린더와 아시안게임 연도 캘린더를 모두 정의.
- 아시안게임 4년 주기 규칙 정의.
- LCK 선수 풀과 LPL/LEC/LTA/LCP 상대 팀 데이터 범위 정의.
- 어빌/포텐/종합점수와 세부 스탯 체계 정의.
- 계약, 나이, 은퇴, 군 입대, 유망주 생성의 최소 규칙 정의.
- AI 도움으로 초기 선수/이벤트 데이터 생성.
- 화면 흐름 초안 작성.
- 커스텀 테마 기능을 MVP 포함/후순위 중 어디에 둘지 결정.
- GitHub issue 생성.

### 2주차

- 로스터 빌더 구현.
- 예산과 포지션 제약 구현.
- 계약 타입 구현: 1년, 2년, 1+1년 옵션.
- 전략/훈련/선수 교체 설정 구현.
- 경기 시뮬레이션 구현.
- 시즌 캘린더와 LCK Split/국제전 진출 흐름 구현.
- 일반 연도/아시안게임 연도 캘린더 전환 로직 구현.
- 순위표와 이벤트 카드 추가.
- 첫 플레이 가능한 시즌 구성.

### 3주차

- 데이터 밸런싱.
- 2-3시즌 연속 플레이 테스트.
- 20시즌 장기 시뮬레이션이 저장 구조상 가능한지 점검.
- UI 폴리싱.
- 가능하면 localStorage 추가.
- 아시안게임 연도 이벤트를 축약 구현.
- 시간이 남으면 커스텀 테마 JSON 가져오기 기능 추가.
- 최종 시즌 요약 화면 추가.
- 배포.
- README/Wiki/회고 작성.

## 4. 판단 메모

강점:

- 게임답고 반복 플레이 가능성이 있습니다.
- 프론트엔드에서 보여줄 화면이 풍부합니다.
- 런타임 AI 없이도 AI 생성 콘텐츠를 개발 과정에서 잘 활용할 수 있습니다.
- 커스텀 테마 시스템으로 개인화와 확장성을 보여줄 수 있습니다.
- LCK 중심 로스터와 국제전 등장 구조로 시즌 서사가 생깁니다.

약점:

- 범위 관리가 필요합니다.
- 시뮬레이션 품질이 밸런싱에 좌우됩니다.
- 실제 LoL/LCK 데이터가 바뀌면 갱신 비용이 생깁니다.
- LCK Split/국제전/아시안게임을 모두 넣으면 MVP가 커질 수 있습니다.
- 커스텀 테마 기능은 구현 우선순위를 잘못 잡으면 핵심 게임 완성도를 해칠 수 있습니다.
